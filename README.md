## About

LAVA is a **sportswear manufacturer and retailer** based in Egypt. The website features:
- **Homepage** — Overview of manufacturing and retail services
- **Factory page** — Custom manufacturing for teams and academies
- **Store page** — Ready-to-purchase product catalog

## Tech Stack

- **Framework:** Next.js 16 (Static Export)
- **Styling:** Tailwind CSS v4
- **Animation:** Framer Motion
- **Deployment:** GitHub Pages
- **Language:** Arabic (RTL) with English support
